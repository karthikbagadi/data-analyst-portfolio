# Sales Analysis Dashboard (SQL + Tableau)

## 📊 Project Overview
This project analyzes sales data using SQL and visualizes insights using Tableau. 
The goal is to understand product performance and sales trends.

---

## 🧠 SQL Analysis
Performed data analysis using MySQL:
- Calculated total sales
- Analyzed sales by product line
- Extracted monthly sales trends

---

## 📈 Tableau Dashboard
Built an interactive dashboard to visualize:
- Sales by Product Line
- Monthly Sales Trend
- Interactive filtering between charts

---

## 🔍 Key Insights
- Identified top-performing product categories
- Observed monthly sales patterns
- Compared product line performance

---

## 🛠 Tools Used
- MySQL (SQL queries)
- Tableau Public (Dashboard)
- CSV Dataset

---

## 📁 Project Files
- `sales_analysis.sql` → SQL queries
- `dashboard.twbx` → Tableau dashboard
